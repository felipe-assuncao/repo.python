To view or run the code, you can:
1. Simply click on the `.ipynb` file right here on GitHub to read the code and see the results.
2. Clone this repository and open the file in an editor like [VS Code](https://code.visualstudio.com/) (with the Jupyter extension) or [Google Colab](https://colab.research.google.com/).
